# 图片重命名工具 - 使用指南

## 这是什么？

一个多功能命名工具，包含两大功能：
1. **图片重命名** - 批量重命名图片，中文自动翻译成英文
2. **命名转换器** - 任意格式文本转换为标准命名格式

## 怎么打开？

双击 `启动.command` 文件即可。

> 首次打开会弹出终端窗口，这是正常的，应用窗口会自动弹出。

---

## 功能一：图片重命名

### 界面说明

#### 顶部按钮栏

| 按钮 | 功能 |
|------|------|
| 选择图片 | 选择一张或多张图片 |
| 选择文件夹 | 选择整个文件夹内的所有图片 |
| 管理类型 | 添加/修改/删除图片类型 |
| 管理模块 | 添加/修改/删除功能模块 |
| 清空 | 清空图片列表 |
| 运行 | 执行重命名 |

#### 图片列表

每一行可以修改：

| 列名 | 说明 |
|------|------|
| 类型 | 下拉选择，如 icon、bg、img、btn |
| 模块 | 下拉选择图片所属的功能模块 |
| 功能名称 | 输入图片用途（支持中文，会自动翻译） |
| 新文件名 | 实时预览最终文件名 |

#### 底部输出目录

- **默认**：空白表示在原目录直接重命名（覆盖原文件）
- **选择**：点击"选择"按钮指定输出目录（复制到新目录）
- **清除**：点击"清除"按钮恢复为原目录模式

### 使用步骤

1. 添加图片（选择图片或选择文件夹）
2. 编辑信息（类型、模块、功能名称）
3. 设置输出目录（可选）
4. 点击「运行」执行重命名

### 命名规则

最终文件名格式：`类型_模块_功能名.扩展名`

例如：
- `icon_home_settings.webp`
- `bg_user_avatar.png`
- `btn_chat_send.webp`

## 功能二：命名转换器

### 这是什么？

输入任意格式的文本（中英文混合、驼峰、下划线等），自动转换为 6 种标准命名格式。

### 使用步骤

1. 在「原文」输入框输入文本（或点击「粘贴」）
2. 自动生成 6 种格式的转换结果
3. 点击结果右侧的「复制」按钮复制使用

### 支持的输入格式

- 中文：首页设置按钮
- 英文混写：homeorderblack
- 驼峰：homeOrderBlack
- 下划线：home_order_black
- 中横线：home-order-black
- 中英混合：首页home设置

### 输出格式

| 格式 | 示例 | 用途 |
|------|------|------|
| 驼峰命名法 | homeOrderBlack | JavaScript 变量 |
| 帕斯卡命名法 | HomeOrderBlack | 类名、组件名 |
| 下划线+小写 | home_order_black | 数据库字段、Python |
| 下划线+大写 | HOME_ORDER_BLACK | 常量 |
| 包名 | home.order.black | Java 包名 |
| 中横线+小写 | home-order-black | CSS 类名、URL |

### 智能识别

工具会自动识别：
- 单词边界（驼峰识别）
- 中文转英文
- 数字分组
- 常见分隔符

例如：`homeOrderBlack2024` → 识别为 `home`, `order`, `black`, `2024`

## 常见问题

**Q: 翻译不准确怎么办？**  
A: 在"功能名称"里直接输入英文，就不会翻译了。

**Q: 命名转换器的翻译准确吗？**  
A: 使用 Google 翻译，单词翻译较准确，复杂句子可能需要手动调整。

**Q: 会覆盖原文件吗？**  
A: 图片重命名时，如果不设置输出目录，会直接重命名原文件。

**Q: 文件名冲突怎么办？**  
A: 工具会自动在文件名后加数字，如 `icon_home_settings_1.webp`。

**Q: 命名转换器支持复制吗？**  
A: 是的，点击每个结果右侧的「复制」按钮即可。
